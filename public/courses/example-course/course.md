# Example Course Template

Welcome to the **Example Course Template**! This course demonstrates all available features in EduHub.

## What You'll Learn

This template showcases:

- **Markdown formatting** for rich content
- **LaTeX equations** for mathematics
- **Code blocks** with syntax highlighting
- **Quizzes** for knowledge testing
- **Flashcards** for memorization
- **Quick reference** for formulas

## How to Use This Template

Copy this `example-course` folder and modify the files to create your own course. Each file type is explained in the README.md file in the courses folder.
