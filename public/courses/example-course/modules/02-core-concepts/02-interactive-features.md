# Interactive Features

EduHub includes several interactive features to enhance the learning experience.

## Flashcards

Flashcards help with **memorization** through active recall. Access them with the `F` key or the flashcard button.

### How Flashcards Work

1. **Front side** shows the question
2. Click to reveal the **back side** with the answer
3. Mark as "Know" or "Don't Know"
4. Cards you struggle with appear more often

### Best Practices

- Review flashcards daily
- Focus on cards you find difficult
- Use short, focused sessions
- Don't rush - think before flipping

## Personal Notes

Take notes within each lesson:

- Press `N` to open the notes panel
- Notes support **markdown** formatting
- Automatically saved to your browser
- Export anytime from settings

### Note-Taking Tips

```markdown
# Use headings to organize
- Bullet points for lists
**Bold** key terms
```

## Progress Tracking

EduHub automatically tracks your learning:

| Feature | Description |
|---------|-------------|
| Completed lessons | Mark lessons as done with `M` |
| Time tracking | Study time is recorded |
| Streaks | Daily learning streaks |
| Quiz scores | Best scores saved |

## Bookmarks

Bookmark important lessons:

- Press `B` to bookmark current lesson
- Access from the home page
- Quickly return to key content

## Keyboard Shortcuts

Speed up your learning with keyboard shortcuts:

| Key | Action |
|-----|--------|
| `←` / `→` | Previous/Next lesson |
| `M` | Mark as complete |
| `Q` | Quick reference |
| `F` | Flashcards |
| `N` | Notes |
| `B` | Bookmark |
| `?` | Show all shortcuts |
| `/` | Search |
| `Esc` | Go back |

## Summary

Interactive features make learning more effective:

- **Flashcards** for memorization
- **Notes** for personal insights
- **Progress** to stay motivated
- **Shortcuts** for efficiency
