# Signal Operations

## Module Overview

This module covers the essential operations that can be performed on signals, both in the amplitude domain and the time domain. Understanding these operations is fundamental to signal processing, as they enable us to modify, analyze, and combine signals for various applications.

## Learning Objectives

By the end of this module, you will be able to:

1. Perform amplitude operations including offset, gain, addition, and multiplication
2. Apply time-domain operations such as shifting, scaling, and reversal
3. Represent sinusoidal signals using amplitude, frequency, and phase parameters
4. Combine multiple operations to achieve desired signal transformations

## Lessons

- **Lesson 1:** Amplitude Operations - Offset, gain, addition, multiplication
- **Lesson 2:** Time Operations - Time shifting, scaling, and reversal
- **Lesson 3:** Sinusoidal Signals - Representation and properties

## Key Concepts

- **Amplitude Operations:** Modify the magnitude values of a signal
- **Time Operations:** Transform the time axis of a signal
- **Sinusoidal Signals:** Fundamental periodic signals used in signal analysis
- **Signal Transformation:** Combining operations to achieve desired results

## Prerequisites

- Understanding of signal definition and classification
- Basic algebra and trigonometry
- Familiarity with function transformations
