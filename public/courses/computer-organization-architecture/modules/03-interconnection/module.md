# Module 3: Computer Function and Interconnection

This module examines how computers execute instructions and how components communicate through interconnection structures. Understanding these concepts is essential for grasping how data flows through a computer system.

## Learning Objectives

By the end of this module, you will be able to:
- Describe the instruction cycle and its phases
- Explain interrupt handling mechanisms
- Compare bus and point-to-point interconnect architectures
- Understand PCI Express technology
