# Module 5: External Memory and I/O Systems

This module covers external storage technologies and I/O mechanisms for data transfer between the computer and external devices.

## Learning Objectives

By the end of this module, you will be able to:
- Describe magnetic disk organization and performance
- Compare different RAID levels
- Explain I/O techniques: programmed I/O, interrupt-driven I/O, and DMA
