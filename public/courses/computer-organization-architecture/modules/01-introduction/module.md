# Module 1: Introduction to Computer Organization

This foundational module establishes the essential concepts and terminology needed throughout the course. Students begin by understanding the distinction between computer organization and architecture, then explore the basic structure and function of computer systems.

## Learning Objectives

By the end of this module, you will be able to:
- Differentiate between computer organization and computer architecture
- Describe the four basic functions of a computer
- Identify main structural components of a computer system
- Convert between decimal, binary, and hexadecimal number systems
