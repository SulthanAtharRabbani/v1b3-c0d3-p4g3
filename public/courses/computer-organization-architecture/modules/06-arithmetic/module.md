# Module 6: Computer Arithmetic

This module covers how computers represent and manipulate numerical data at the hardware level, including integer and floating-point arithmetic.

## Learning Objectives

By the end of this module, you will be able to:
- Understand signed and unsigned integer representations
- Perform binary arithmetic operations
- Understand IEEE 754 floating-point representation
- Describe the structure and function of the ALU
